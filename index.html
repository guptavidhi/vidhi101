About this Page

  • The Design  

    The whole page is built to emulate the Visual Studio Code's "Abyss" theme and the entire thing is made from scratch. The symbols and icons used throughout the page are from FontAwesome and Material Design Iconic Font.
     

  • The Build

    The technology used is simple HTML, CSS and JavaScript along with Bootstrap framework.
    It's hosted by AWS EC2 with Amazon Route 53 as the DNS.
    The domain name is registered at Namecheap.
    It's also created with a lot of sweat and tears and sleepless nights.

    This page can be treated as a Creative CV or a personal blog of sorts but please don't use it as either. It is truly just a motivation to learn some Web Development.
